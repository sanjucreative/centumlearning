# centumlearning
Centum Learning has provided customized end-to-end training solutions


# Requirements #

* `NPM`
* `GULP CLI`

# HOW TO RUN #

* Clone repository
* Run `npm install`
* Run `npm start` - for development
* Run `npm build` - for production 
* Copy the generated `assets` folder.


# Gulp Start $

* Run 'gulp serve'
